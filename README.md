# Érettségi Történelem – Középszint (Markdown projekt)

## Mi ez a projekt?

Ez a mappa egy olyan, középszintű történelem érettségire felkészítő tanulási projekt, amely korszakonként és témánként rendezett Markdown fájlokból áll.

## Mappastruktúra és korszakok

A gyökérmappa neve: `erot_tortenelem_kozep`. Ezen belül az egyes történelmi korszakok külön almappákban találhatók.

- `05_ujkor` – **Újkor – ipari forradalmak, nemzetállamok, világháborúk**

## Hogyan használd tanuláshoz?

1. Válaszd ki azt a korszakot, amelyből éppen ismételni szeretnél.
2. Először olvasd el az `index.md` fájlt.
3. Ezután nyisd meg a részletes témákat tartalmazó `.md` fájlokat, és jegyzetelj melléjük.
4. Különösen a kulcsfogalmak szakasz ismétlésére érdemes időt szánni.
